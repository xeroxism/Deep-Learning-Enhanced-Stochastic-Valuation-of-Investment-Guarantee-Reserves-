# Deep Learning Enhanced Stochastic Valuation of Investment Guarantee Reserves Under Nigerian Market Volatility

This repository contains the official code, data structures, and documentation for the research project titled **"Deep Learning Enhanced Stochastic Valuation of Investment Guarantee Reserves Under Nigerian Market Volatility"**, submitted as a Capstone Project at **WorldQuant University (WQU)** and prepared for journal publication.

## 📝 Project Overview
This research introduces a novel framework that marries traditional actuarial stochastic modeling with advanced Deep Learning (DL) architectures. The primary objective is to enhance the valuation of investment guarantee reserves (such as Minimum Guaranteed Maturity Benefits) within the highly volatile Nigerian financial market. 

By leveraging deep learning to capture non-linear market dynamics and regime shifts, this project provides a more robust, computationally efficient, and risk-adjusted approach to determining capital reserve requirements for financial institutions operating in emerging markets.
